# AI Recruiting Agent — Knowledge Base

This repository contains the knowledge base for an AI agent that assists with recruiting and talent acquisition.

## Structure

```
knowledge-base/
├── company/
│   ├── overview.md          # Mission, vision, values
│   ├── products.md          # What we build and why
│   └── culture.md           # How we work, what we believe in
├── team/
│   ├── structure.md         # Teams, roles, reporting lines
│   ├── engineering.md       # Engineering org specifics
│   └── hiring-managers.md   # Key contacts per role/team
├── tone-of-voice/
│   ├── guidelines.md        # How we communicate with candidates
│   ├── examples.md          # Good vs. bad message examples
│   └── templates.md         # Reusable message templates
└── recruiting/
    ├── process.md            # End-to-end hiring process
    ├── scorecards.md         # Interview criteria and evaluation
    ├── sourcing.md           # Where and how we source candidates
    └── best-practices.md     # Dos and don'ts
```

## Usage

The agent uses these files as its primary context when helping with:

- Writing outreach messages to candidates
- Evaluating candidate profiles
- Answering questions about open roles
- Preparing interview questions
- Drafting job descriptions
